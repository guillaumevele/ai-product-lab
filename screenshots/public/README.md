# Public Screenshots

This folder is reserved for reviewed, public-safe screenshots only.

Rules:

- Synthetic content only, or fully redacted screenshots.
- No patient, user or client data.
- No private product roadmap.
- No production secrets, URLs or internal identifiers.
- No medical or legal conclusions in visible UI unless explicitly reviewed.
- Strip EXIF, device, GPS, creator and editing-history metadata before commit.

Screenshots outside this folder are ignored by default.
